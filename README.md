# Blog Privado - Escape Room

Página estática para escape room que simula un blog privado de una influencer.

## Estructura del proyecto

```
webeme_2/
├── index.html          # Página principal
├── styles.css          # Estilos
├── script.js           # Lógica de la aplicación
├── assets/
│   └── video.mp4      # Video para la sección "Desvelo la verdad"
└── README.md           # Este archivo
```

## Funcionalidades

- **Login**
- **Noticias encubiertas**: 5 noticias sobre Valenthia
- **Desvelo la verdad**: Solo accesible después de leer al menos 1 noticia
- **Elección final**: Dos opciones con consecuencias diferentes

## Personalización

Puedes modificar:
- El contenido de las noticias en `index.html`
- Los estilos en `styles.css`
- La lógica en `script.js`
- El video reemplazando `assets/video.mp4`
